### エラー

```
xxx.shader forced to unroll loop, but unrolling failed.
```

## エラー分析

- カメラ手前のものしかぼかしされない、遠いものをぼかし必要がない
- frag内でループ回数はcompile時で決まらなくて展開できない
- ディフォルト値は[unroll]
- 今回実行中でループ回数が決まる場合は[loop]が適正です

## 修正した仮コード

{
	if(i.blur)
	{
        //ボカしの計算

            }
        }
        return color;
	}
	return tex2D(_MainTex, i.uv);
